방화벽(based on Windows XP) 구현

요약

Window XP 기반의 방화벽으로써 방화벽의 기본기능 (모니터링, 블로킹, 세션 관리, 금지어 차단)을 구현한다. 필요시 SMS로 사용자에게 알림 문자를 전송하며 원격 제어로 기본 기능을 이용할 수 있다.

주요 기능

1. Traffic Monitoring

현재 네트워크에 접속중인 프로세스의 상태 정보 보기

호스트가 사용하고 있는 모든 Port 목록 출력

현재 네트워크에 접속중인 프로세스의 상태 정보 보기 - 호스트가 사용하고 있는 모든 Port 목록 출력 - 각 Port 의 네트워크 Traffic 총 송수신량을 Byte 단위로 표현 - 출력은 텍스트 또는 그래프 등을 사용하여 시각화

각 Port 의 네트워크 Traffic 총 송수신량을 Byte 단위로 표현 - 출력은 텍스트 또는 그래프 등을 사용하여 시각화

2. Traffic Blocking

입력 받은 IP주소(필수) 및 Port 번호(선택)를 차단

Traffic Blocking 활성화/비활성화 - Port 번호가 입력되지 않은 겅우 해당 IP 주소에 대한 호스트와 수행하는 모든 Traffic 차단

Port 번호가 입력되지 않은 겅우 해당 IP 주소에 대한 호스트와 수행하는 모든 Traffic 차단

3. TCP Session

초기값은 TCP Session 수를 0으로 설정

최대 허용 TCP Session 수 제어

현재 사용중인 TCP Session의 수를 실시간으로 감지

허용 가능한 TCP Session의 수를 초과하는 연결시도 차단

4. 금지 단어 검출(UDP)

모든 송수신 패킷을 검사하여 입력 받은 텍스트가 포함된 패킷 버림

UDP 사용, 입력은 충분히 짧은 텍스트로 제한

5. 원격 제어

다른 컴퓨터 시스템에서 방화벽에 접근하여 여러 기능들을 제어(원격 제어 프로그램이 따로 존재하고, 이를 통해 방화벽 시스템에 접근 가능)
