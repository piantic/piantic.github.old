웹 카메라를 이용한 증강현실 구현

요약

Ogre3D 엔진, OpenCV, 그리고 영상처리 알고리즘을 이용하여 증강현실을 구현한다.

1. Overview

영상처리에서 템플릿 매치 기법을 이용하여 얼굴을 검출하였다. 템플릿 매치 기법을 하기위해서는 미리 얼굴영상에 대한 data가 필요하다. 이 data는 xml 형태로 작성되어있다. 얼굴을 검출하기 위해 프로그램에서는 xml 파일의 내용을 가져와서 분석, 현재영상에 얼굴이 있는지 검사한다.

얼굴이 검출되었으면, 얼굴이 검출된 영역의 무게중심을 구한다. 무게중심에 미리 정의되어있는 3d object를 얼굴 영역의 반지름에 맞추어 적당히 크기를 바꾼다. 계산된 값을 이용하여 얼굴에 3d object를 표현한다.

2. 거리인식

웹 카메라와 얼굴의 거리에 따라서 3d object의 크기가 커진다. 일정한 거리에서는 3d object 뿐만 아니라 particle 효과도 같이 화면에 표현되도록 하였다.

3. 물체 인식

특정 물체의 색깔을 인식하여 화면에서 물체가 있는 위치를 파악하였다. 해당 물체에 graphic 요소를 첨가함으로써 물체가 인식 되었다는 사실을 알 수 있다. 구현을 쉽게 하기 위해서 인식하는 물체의 색깔은 빨간색으로 한정하였다.

성능과 기능에 대한 분석

현재 만들어진 시스템은 실시간으로 영상을 분석하여 적절하게 graphic 요소를 표현할 수 있다. 인식 가능한 색깔 확장, 물체의 외곽선 검출, 다양한 3d object 추가와 같은 부분이 시스템에 반영된다면 더욱 더 멋진 증강현실 시스템을 만들 수 있으리라고 생각한다.

실시간으로 영상에서 대상을 인식하고 처리하는 부분에서 약간의 문제가 있다. 얼굴 인식이나 색깔 인식이 항상 제대로 되는 것은 아니기 때문이다. 기본적인 인식에서 오류가 발생하면 graphic 요소가 화면에 잘못 반영될 수 있다.
